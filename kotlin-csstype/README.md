# kotlin-csstype

[CSSType](https://github.com/frenic/csstype) - TypeScript and Flow definitions for CSS, generated by [data from MDN](https://github.com/mdn/data).
*kotlin-csstype* library is *CSSType* adoption for *Kotlin*.
