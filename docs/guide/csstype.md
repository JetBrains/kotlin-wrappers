# FAQ

## How to avoid name intersections while using `csstype` values?

We recommend to apply static imports strictly for next entities:
- `AnimationIterationCount`
- `AnimationFillMode`
- `AnimationTimingFunction`
- `Auto`
- `AutoRepeat`
- `BoxSizing`
- `Globals`
- `Length`
- `LineStyle`
- `NamedColor`
- `None`
- `Round`
- `TransitionTimingFunction`
- `Visibility`
